# MyBlog

> A minimal personal blog built with Next.js 16 + MDX, focused on content-first writing experience.

## Features

- ✍️ **MDX Content**: Write blog posts in Markdown with JSX support
- 📅 **Date Handling**: date-fns for formatted post dates
- 🎨 **Dark Mode**: System-aware light/dark theme
- ⚡ **Static Generation**: Fast page loads with Next.js SSG

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Content**: MDX + gray-matter + remark-html
- **Styling**: Tailwind CSS 4
- **Deploy**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
